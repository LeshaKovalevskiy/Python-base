## 2.–ü–µ—Ä–µ–º–µ–Ω–Ω—ã–µ, –æ–ø–µ—Ä–∞—Ç–æ—Ä –ø—Ä–∏—Å–≤–∞–∏–≤–∞–Ω–∏—è, —Ñ—É–Ω–∫—Ü–∏–∏ type –∏ id

**–ü–µ—Ä–µ–º–µ–Ω–Ω–∞—è** - –∏–º–µ–Ω–æ–≤–∞–Ω–Ω–∞—è –æ–±–ª–∞—Å—Ç—å –ø–∞–º—è—Ç–∏, –ø—Ä–µ–¥–Ω–∞–∑–Ω–∞—á–µ–Ω–Ω–∞—è –¥–ª—è —Ö—Ä–∞–Ω–µ–Ω–∏—è –∑–Ω–∞—á–µ–Ω–∏—è. –ü–µ—Ä–µ–º–µ–Ω–Ω—É—é –º–æ–∂–Ω–æ –ø—Ä–µ–¥—Å—Ç–∞–≤–∏—Ç—å –≤ –≤–∏–¥–µ –∫–æ—Ä–æ–±–æ—á–∫–∏, –≤ –∫–æ—Ç–æ—Ä–æ–π –º–æ–∂–Ω–æ —Ö—Ä–∞–Ω–∏—Ç—å –æ–¥–Ω–æ 
–∑–Ω–∞—á–µ–Ω–∏–µ. –ó–Ω–∞—è –∏–º—è –∫–æ—Ä–æ–±–∫–∏, –≤—ã –≤—Å–µ–≥–¥–∞ –º–æ–∂–µ—Ç–µ –æ–±—Ä–∞—Ç–∏—Ç—å—Å—è –∫ –µ–µ —Å–æ–¥–µ—Ä–∂–∏–º–æ–º—É –∏–ª–∏ –∑–∞–º–µ–Ω–∏—Ç—å –µ–≥–æ –¥—Ä—É–≥–∏–º –∑–Ω–∞—á–µ–Ω–∏–µ–º.

<h3 align="center">–ò–º—è –ø–µ—Ä–µ–º–µ–Ω–Ω–æ–π</h3>

**–†–µ–∫–æ–º–µ–Ω–¥–∞—Ç–µ–ª—å–Ω–æ–µ –ø—Ä–∞–≤–∏–ª–æ**

> –°—É—â–µ—Å—Ç–≤—É–µ—Ç –æ–±—â–µ–ø—Ä–∏–Ω—è—Ç–æ–µ –ø—Ä–∞–≤–∏–ª–æ: –Ω–µ –∏—Å–ø–æ–ª—å–∑–æ–≤–∞—Ç—å —Ç—Ä–∞–Ω—Å–ª–∏—Ç –¥–ª—è –∏–º—ë–Ω, —Ç–æ–ª—å–∫–æ –∞–Ω–≥–ª–∏–π—Å–∫–∏–π —è–∑—ã–∫. –°—Ä–µ–¥–∏ —Ä–∞–∑—Ä–∞–±–æ—Ç—á–∏–∫–æ–≤ –µ—Å—Ç—å —à—É—Ç–∫–∞: ¬´—Å–∞–º–æ–µ —Å–ª–æ–∂–Ω–æ–µ –≤ –ø—Ä–æ–≥—Ä–∞–º–º–∏—Ä–æ–≤–∞–Ω–∏–∏ ‚Äî –Ω–∞–∑–≤–∞–Ω–∏—è –ø–µ—Ä–µ–º–µ–Ω–Ω—ã—Ö –∏ –∏–Ω–≤–∞–ª–∏–¥–∞—Ü–∏—è –∫–µ—à–∞¬ª.

### –ó–∞–¥–∞–Ω–∏—è

1. –í –Ω–µ–∫–æ—Ç–æ—Ä–æ–π —à–∫–æ–ª–µ —Ä–µ—à–∏–ª–∏ –Ω–∞–±—Ä–∞—Ç—å —Ç—Ä–∏ –Ω–æ–≤—ã—Ö –º–∞—Ç–µ–º–∞—Ç–∏—á–µ—Å–∫–∏—Ö –∫–ª–∞—Å—Å–∞ –∏ –æ–±–æ—Ä—É–¥–æ–≤–∞—Ç—å –∫–∞–±–∏–Ω–µ—Ç—ã –¥–ª—è –Ω–∏—Ö –Ω–æ–≤—ã–º–∏ –ø–∞—Ä—Ç–∞–º–∏. 
–ó–∞ –∫–∞–∂–¥–æ–π –ø–∞—Ä—Ç–æ–π –º–æ–∂–µ—Ç —Å–∏–¥–µ—Ç—å –¥–≤–∞ —É—á–∞—â–∏—Ö—Å—è. –ò–∑–≤–µ—Å—Ç–Ω–æ –∫–æ–ª–∏—á–µ—Å—Ç–≤–æ —É—á–∞—â–∏—Ö—Å—è –≤ –∫–∞–∂–¥–æ–º –∏–∑ —Ç—Ä–µ—Ö –∫–ª–∞—Å—Å–æ–≤. –í—ã–≤–µ–¥–∏—Ç–µ –Ω–∞–∏–º–µ–Ω—å—à–µ–µ 
—á–∏—Å–ª–æ –ø–∞—Ä—Ç, –∫–æ—Ç–æ—Ä–æ–µ –Ω—É–∂–Ω–æ –ø—Ä–∏–æ–±—Ä–µ—Å—Ç–∏ –¥–ª—è –Ω–∏—Ö.

2. –î–∞–Ω—ã —Ç—Ä–∏ –≤–µ—â–µ—Å—Ç–≤–µ–Ω–Ω—ã—Ö —á–∏—Å–ª–∞ a, b, c. –ù–∞–ø–∏—à–∏—Ç–µ –ø—Ä–æ–≥—Ä–∞–º–º—É, –∫–æ—Ç–æ—Ä–∞—è –Ω–∞—Ö–æ–¥–∏—Ç –≤–µ—â–µ—Å—Ç–≤–µ–Ω–Ω—ã–µ –∫–æ—Ä–Ω–∏ –∫–≤–∞–¥—Ä–∞—Ç–Ω–æ–≥–æ —É—Ä–∞–≤–Ω–µ–Ω–∏—è ax^2 + bx + c = 0. 

–ù–∞ –≤—Ö–æ–¥ –ø—Ä–æ–≥—Ä–∞–º–º–µ –ø–æ–¥–∞–µ—Ç—Å—è —Ç—Ä–∏ –≤–µ—â–µ—Å—Ç–≤–µ–Ω–Ω—ã—Ö —á–∏—Å–ª–∞ a, b, c –∫–∞–∂–¥–æ–µ –Ω–∞ –æ—Ç–¥–µ–ª—å–Ω–æ–π —Å—Ç—Ä–æ–∫–µ.

–ü—Ä–æ–≥—Ä–∞–º–º–∞ –¥–æ–ª–∂–Ω–∞ –≤—ã–≤–µ—Å—Ç–∏ –≤–µ—â–µ—Å—Ç–≤–µ–Ω–Ω—ã–µ –∫–æ—Ä–Ω–∏ —É—Ä–∞–≤–Ω–µ–Ω–∏—è –µ—Å–ª–∏ –æ–Ω–∏ —Å—É—â–µ—Å—Ç–≤—É—é—Ç –∏–ª–∏ —Ç–µ–∫—Å—Ç ¬´–ù–µ—Ç –∫–æ—Ä–Ω–µ–π¬ª –≤ –ø—Ä–æ—Ç–∏–≤–Ω–æ–º —Å–ª—É—á–∞–µ.

–ü—Ä–∏–º–µ—á–∞–Ω–∏–µ. –ï—Å–ª–∏ —É—Ä–∞–≤–Ω–µ–Ω–∏–µ –∏–º–µ–µ—Ç –¥–≤–∞ –∫–æ—Ä–Ω—è, —Ç–æ —Å–ª–µ–¥—É–µ—Ç –≤—ã–≤–µ—Å—Ç–∏ –∏—Ö –≤ –ø–æ—Ä—è–¥–∫–µ –≤–æ–∑—Ä–∞—Å—Ç–∞–Ω–∏—è.

```python
a, b, c = float(input()), float(input()), float(input())

if D > 0:
    x1 = (-b + D ** 0.5) / (2 * a)
    x2 = (-b - D ** 0.5) / (2 * a)
    print(min(x1, x2), max(x1, x2), sep="\n")
elif D == 0:
    print(-b / (2 * a))
else:
```

–î–∞–≤–Ω–æ —Ö–æ–¥—è—Ç —Å–ª—É—Ö–∏, —á—Ç–æ –±—Ä–∏–≥–∞–¥–∏—Ä –≤ –¥—è–¥—é—à–∫–∏–Ω–æ–π —Ñ–∏—Ä–º–µ –ø–æ–∫—É–ø–∞–µ—Ç –ª–∏—à–Ω–µ–µ –∫–æ–ª–∏—á–µ—Å—Ç–≤–æ —Å—Ç—Ä–æ–π–º–∞—Ç–µ—Ä–∏–∞–ª–æ–≤, –∞ –æ—Å—Ç–∞—Ç–∫–∏ –∏—Å–ø–æ–ª—å–∑—É–µ—Ç

—Å—Ç–µ–Ω –≤ –æ—Ñ–∏—Å–µ –¥–ª–∏–Ω–æ–π L –º–µ—Ç—Ä–æ–≤, —à–∏—Ä–∏–Ω–æ–π ‚Äì W –∏ –≤—ã—Å–æ—Ç–æ–π ‚Äì H, –µ—Å–ª–∏ –æ–¥–Ω–æ–π –±–∞–Ω–∫–∏ —Ö–≤–∞—Ç–∞–µ—Ç –Ω–∞ 16–º2 , –∞ —Ä–∞–∑–º–µ—Ä–∞–º–∏ –¥–≤–µ—Ä–µ–π –∏ –æ–∫–æ–Ω

–ü–æ —É–º–æ–ª—á–∞–Ω–∏—é –∫–æ–º–∞–Ω–¥–∞ `print()` –ø—Ä–∏–Ω–∏–º–∞–µ—Ç –Ω–µ—Å–∫–æ–ª—å–∫–æ –∞—Ä–≥—É–º–µ–Ω—Ç–æ–≤ (–ø–∞—Ä–∞–º–µ—Ç—Ä–æ–≤), –≤—ã–≤–æ–¥–∏—Ç –∏—Ö —á–µ—Ä–µ–∑ –æ–¥–∏–Ω **–ø—Ä–æ–±–µ–ª**, –ø–æ—Å–ª–µ —á–µ–≥–æ **—Å—Ç–∞–≤–∏—Ç –ø–µ—Ä–µ–≤–æ–¥ —Å—Ç—Ä–æ–∫–∏**. –≠—Ç–æ –ø–æ–≤–µ–¥–µ–Ω–∏–µ

<h3 align="center">–ü–∞—Ä–∞–º–µ—Ç—Ä sep</h3>

–ü—Ä–∏ –ø–µ—Ä–≤–æ–π –ø–µ—á–∞—Ç–∏ —Ä–∞–∑–¥–µ–ª–∏—Ç–µ–ª—å –ø–æ —É–º–æ–ª—á–∞–Ω–∏—é `sep=' '`.

–ï—Å–ª–∏ –ø–µ—Ä–µ–≤–æ–¥ —Å—Ç—Ä–æ–∫–∏ –¥–µ–ª–∞—Ç—å –Ω–µ –Ω—É–∂–Ω–æ –∏–ª–∏ —Ç—Ä–µ–±—É–µ—Ç—Å—è —É–∫–∞–∑–∞—Ç—å —Å–ø–µ—Ü–∏–∞–ª—å–Ω–æ–µ –æ–∫–æ–Ω—á–∞–Ω–∏–µ, —Ç–æ —Å–ª–µ–¥—É–µ—Ç —è–≤–Ω–æ —É–∫–∞–∑–∞—Ç—å –∑–Ω–∞—á–µ–Ω–∏–µ –¥–ª—è –ø–∞—Ä–∞–º–µ—Ç—Ä–∞ `end`.

–†–∞—Å—Å–º–æ—Ç—Ä–∏–º —Å–ª–µ–¥—É—é—â–∏–π –∫–æ–¥:

–†–∞—Å—Å–º–æ—Ç—Ä–∏–º —Å–ª–µ–¥—É—é—â–∏–π –∫–æ–¥:

## 6.–õ–æ–≥–∏—á–µ—Å–∫–∏–π —Ç–∏–ø Bool. –û–ø–µ—Ä–∞—Ç–æ—Ä—ã —Å—Ä–∞–≤–Ω–µ–Ω–∏—è 

<h3 align="center">–õ–æ–≥–∏—á–µ—Å–∫–∏–π —Ç–∏–ø –¥–∞–Ω–Ω—ã—Ö</h3>

–õ–æ–≥–∏—á–µ—Å–∫–∏–π —Ç–∏–ø –¥–∞–Ω–Ω—ã—Ö (–±—É–ª–µ–≤ —Ç–∏–ø, `Boolean`) ‚Äî –ø—Ä–∏–º–∏—Ç–∏–≤–Ω—ã–π —Ç–∏–ø –¥–∞–Ω–Ω—ã—Ö –≤ –∏–Ω—Ñ–æ—Ä–º–∞—Ç–∏–∫–µ, –ø—Ä–∏–Ω–∏–º–∞—é—â–∏–π –¥–≤–∞ –≤–æ–∑–º–æ–∂–Ω—ã—Ö –∑–Ω–∞—á–µ–Ω–∏—è, –∏–Ω–æ–≥–¥–∞ –Ω–∞–∑—ã–≤–∞–µ–º—ã—Ö –∏—Å—Ç–∏–Ω–æ–π (`True`) –∏
–ª–æ–∂—å—é (`False`). –ü—Ä–∏—Å—É—Ç—Å—Ç–≤—É–µ—Ç –≤ –ø–æ–¥–∞–≤–ª—è—é—â–µ–º –±–æ–ª—å—à–∏–Ω—Å—Ç–≤–µ —è–∑—ã–∫–æ–≤ –ø—Ä–æ–≥—Ä–∞–º–º–∏—Ä–æ–≤–∞–Ω–∏—è –∫–∞–∫ —Å–∞–º–æ—Å—Ç–æ—è—Ç–µ–ª—å–Ω–∞—è —Å—É—â–Ω–æ—Å—Ç—å –∏–ª–∏ —Ä–µ–∞–ª–∏–∑—É–µ—Ç—Å—è —á–µ—Ä–µ–∑ —á–∏—Å–ª–µ–Ω–Ω—ã–π —Ç–∏–ø –¥–∞–Ω–Ω—ã—Ö. –ó–∞ 
–∑–Ω–∞—á–µ–Ω–∏–µ "–∏—Å—Ç–∏–Ω–∞" –ø—Ä–∏–Ω–∏–º–∞–µ—Ç—Å—è 1, –∑–∞ –∑–Ω–∞—á–µ–Ω–∏–µ "–ª–æ–∂—å" ‚Äî 0.

<h3 align="center">–û–ø–µ—Ä–∞—Ç–æ—Ä—ã —Å—Ä–∞–≤–Ω–µ–Ω–∏—è</h3>

|–û–ø–µ—Ä–∞—Ç–æ—Ä  | –û–±–æ–∑–Ω–∞—á–µ–Ω–∏–µ  |
| ------------- | ------------- |
| `<` | —Å—Ä–∞–≤–Ω–µ–Ω–∏–µ –Ω–∞ –º–µ–Ω—å—à–µ |
| `>` | —Å—Ä–∞–≤–Ω–µ–Ω–∏–µ –Ω–∞ –±–æ–ª—å—à–µ |
| `<=` | —Å—Ä–∞–≤–Ω–µ–Ω–∏–µ –Ω–∞ –º–µ–Ω—å—à–µ –ª–∏–±–æ —Ä–∞–≤–Ω–æ |
| `>=` | —Å—Ä–∞–≤–Ω–µ–Ω–∏–µ –Ω–∞ –±–æ–ª—å—à–µ –ª–∏–±–æ —Ä–∞–≤–Ω–æ |
| `==` | —Å—Ä–∞–≤–Ω–µ–Ω–∏–µ –Ω–∞ —Ä–∞–≤–µ–Ω—Å—Ç–≤–æ |
| `!=` | —Å—Ä–∞–≤–Ω–µ–Ω–∏–µ –Ω–∞ –Ω–µ—Ä–∞–≤–µ–Ω—Å—Ç–≤–æ |

<h3 align="center">–õ–æ–≥–∏—á–µ—Å–∫–∏–π —Ç–∏–ø –¥–∞–Ω–Ω—ã—Ö –≤ Python</h3>

–õ–æ–≥–∏—á–µ—Å–∫–∏–µ –∑–Ω–∞—á–µ–Ω–∏—è `True` (–∏—Å—Ç–∏–Ω–∞) –∏ `False` (–ª–æ–∂—å) –ø—Ä–µ–¥—Å—Ç–∞–≤–ª—è—é—Ç —Ç–∏–ø –¥–∞–Ω–Ω—ã—Ö `bool`.

–ù–∞–ø—Ä–∏–º–µ—Ä:

```python

```

<h3 align="center">–õ–æ–≥–∏—á–µ—Å–∫–∏–µ –æ–ø–µ—Ä–∞—Ç–æ—Ä—ã –≤ Python</h3>

–î–ª—è —Å–æ–∑–¥–∞–Ω–∏—è –ø—Ä–æ–∏–∑–≤–æ–ª—å–Ω–æ —Å–ª–æ–∂–Ω—ã—Ö –ª–æ–≥–∏—á–µ—Å–∫–∏—Ö –≤—ã—Ä–∞–∂–µ–Ω–∏–π (—É—Å–ª–æ–≤–∏–π) –∏—Å–ø–æ–ª—å–∑—É—é—Ç—Å—è —Ç—Ä–∏ –ª–æ–≥–∏—á–µ—Å–∫–∏–µ –æ–ø–µ—Ä–∞—Ü–∏–∏:

 - –∏ (`and`);
 - –∏–ª–∏ (`or`);
 - –Ω–µ (`not`).
 
–õ–æ–≥–∏—á–µ—Å–∫–∏–µ –æ–ø–µ—Ä–∞—Ü–∏–∏ –∏—Å–ø–æ–ª—å–∑—É—é—Ç –æ–ø–µ—Ä–∞–Ω–¥—ã —Å–æ –∑–Ω–∞—á–µ–Ω–∏—è–º–∏ `True` –∏ `False` –∏ –≤–æ–∑–≤—Ä–∞—â–∞—é—Ç —Ä–µ–∑—É–ª—å—Ç–∞—Ç —Ç–∞–∫–∂–µ —Å –ª–æ–≥–∏—á–µ—Å–∫–∏–º–∏ –∑–Ω–∞—á–µ–Ω–∏—è–º–∏. –û–ø—Ä–µ–¥–µ–ª—ë–Ω–Ω—ã–µ –¥–ª—è –æ–±—ä–µ–∫—Ç–æ–≤ —Ç–∏–ø–∞ `bool`
–æ–ø–µ—Ä–∞—Ç–æ—Ä—ã (`and`, `or`, `not`) –∏–∑–≤–µ—Å—Ç–Ω—ã –∫–∞–∫ –ª–æ–≥–∏—á–µ—Å–∫–∏–µ –æ–ø–µ—Ä–∞—Ç–æ—Ä—ã –∏ –∏–º–µ—é—Ç –æ–±—â–µ–∏–∑–≤–µ—Å—Ç–Ω—ã–µ –æ–ø—Ä–µ–¥–µ–ª–µ–Ω–∏—è:

 - `a and b` –¥–∞—ë—Ç `True`, –µ—Å–ª–∏ –æ–±–∞ –æ–ø–µ—Ä–∞–Ω–¥–∞ `True`, –∏ `False`, –µ—Å–ª–∏ —Ö–æ—Ç—è –±—ã –æ–¥–∏–Ω –∏–∑ –Ω–∏—Ö `False`;
 - `a or b` –¥–∞—ë—Ç `False`, –µ—Å–ª–∏ –æ–±–∞ –æ–ø–µ—Ä–∞–Ω–¥–∞ `False`, –∏ `True`, –µ—Å–ª–∏ —Ö–æ—Ç—è –±—ã –æ–¥–∏–Ω –∏–∑ –Ω–∏—Ö `True`;
 - `not a` –¥–∞—ë—Ç `True`, –µ—Å–ª–∏ `a` –∏–º–µ–µ—Ç –∑–Ω–∞—á–µ–Ω–∏–µ `False`, –∏ `False`, –µ—Å–ª–∏ `a` –∏–º–µ–µ—Ç –∑–Ω–∞—á–µ–Ω–∏–µ `True`.

<h3 align="center">–ë—É–ª–µ–≤—ã –∑–Ω–∞—á–µ–Ω–∏—è –∫–∞–∫ —á–∏—Å–ª–∞</h3>

–õ–æ–≥–∏—á–µ—Å–∫–∏–µ –∑–Ω–∞—á–µ–Ω–∏—è –≤ Python –º–æ–∂–Ω–æ —Ç—Ä–∞–∫—Ç–æ–≤–∞—Ç—å –∫–∞–∫ —á–∏—Å–ª–∞. –ó–Ω–∞—á–µ–Ω–∏—é `True` —Å–æ–æ—Ç–≤–µ—Ç—Å—Ç–≤—É–µ—Ç —á–∏—Å–ª–æ 1, –≤ —Ç–æ –≤—Ä–µ–º—è –∫–∞–∫ –∑–Ω–∞—á–µ–Ω–∏—é `False` —Å–æ–æ—Ç–≤–µ—Ç—Å—Ç–≤—É–µ—Ç 0. –¢–∞–∫–∏–º –æ–±—Ä–∞–∑–æ–º –º—ã
–º–æ–∂–µ–º —Å—Ä–∞–≤–Ω–∏—Ç—å –ª–æ–≥–∏—á–µ—Å–∫–∏–µ –∑–Ω–∞—á–µ–Ω–∏—è —Å —á–∏—Å–ª–∞–º–∏:


–ú—ã –º–æ–∂–µ–º —Ç–∞–∫–∂–µ –ø—Ä–∏–º–µ–Ω—è—Ç—å –∞—Ä–∏—Ñ–º–µ—Ç–∏—á–µ—Å–∫–∏–µ –æ–ø–µ—Ä–∞—Ü–∏–∏ –∫ –ª–æ–≥–∏—á–µ—Å–∫–∏–º –∑–Ω–∞—á–µ–Ω–∏—è–º.

–ú–æ–∂–Ω–æ –∏—Å–ø–æ–ª—å–∑–æ–≤–∞—Ç—å –¥–ª—è –ø–æ–¥—Å—á–µ—Ç–∞ –∫–æ–ª–∏—á–µ—Å—Ç–≤–∞ —ç–ª–µ–º–µ–Ω—Ç–æ–≤ —É–¥–æ–≤–ª–µ—Ç–≤–æ—Ä—è—é—â–∏—Ö —É—Å–ª–æ–≤–∏—é:

 - —Å—Ç—Ä–æ–∫–∏: –ø—É—Å—Ç–∞—è —Å—Ç—Ä–æ–∫–∞ ‚Äî –ª–æ–∂—å (`False`), –Ω–µ–ø—É—Å—Ç–∞—è —Å—Ç—Ä–æ–∫–∞ ‚Äî –∏—Å—Ç–∏–Ω–∞ (`True`);
